# Paper

Conference paper draft: *When Do Physics Priors Help? A Controlled Benchmark of
Physics-Informed Dynamics Models for Underactuated Swing-Up.*

## Files
- `main.tex` — the paper source (IEEEtran conference class).
- `main.pdf` — the compiled PDF.
- `figures/` — figures used in the paper (copied from `../paper_assets/figures/`,
  which are regenerated by `scripts/aggregate_results.py` and
  `scripts/aggregate_control.py`).

## How to compile

**Overleaf (easiest):** create a new project and upload the `paper/` folder
(`main.tex` plus the `figures/` folder), then click *Recompile*.

**Locally** (needs a TeX distribution such as MiKTeX or TeX Live):
```bash
cd paper
pdflatex main.tex
pdflatex main.tex   # run twice so cross-references resolve
```

The numbers in the tables come from the experiments in this repository; see
`paper_assets/RESULTS.md` and `paper_assets/summary.csv` for the source values.
